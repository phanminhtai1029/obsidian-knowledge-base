> - **App Insights là gì**: APM (Application Performance Monitoring) thuộc Azure Monitor; thu telemetry tự động (request, dependency, exception, page view) + custom.
> - **Instrument**: connection string / instrumentation key; auto-instrumentation (codeless) vs SDK; OpenTelemetry.
> - **Telemetry types**: requests, dependencies, exceptions, traces, custom events/metrics; **distributed tracing** (operation ID liên kết qua nhiều service).
> - **Phân tích**: Application Map, Live Metrics, Failures/Performance blade, **Log Analytics + KQL** để query, **alert** & smart detection.
> - **Sampling** (giảm chi phí/khối lượng telemetry); data retention.
