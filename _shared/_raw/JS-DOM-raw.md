## DOM manipulation performance tips

Let's go over some performance tips when working with a DOM in JavaScript. The first efficiency tip
that can be easily implemented is to save the reference to the element in the DOM after selecting
it, instead of getting it from the page again. Selecting elements from the page is relatively heavy,
so limiting it helps. Here we have a dynamically generated list. You can see on line 12, we have an

createElement or properties such as textContent and innerText. These last two are only possible for
plain text, and often not a real alternative when the HTML needs to contain other elements. The
