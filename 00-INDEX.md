## 🗂️ Quy ước vault

- Mỗi domain/section có 1 file `00-MOC-*` (Map of Content).
- `_shared/` chứa cheatsheets, templates, assets dùng chung.
- Note có frontmatter: `title / section / tags / related / difficulty / estimated_time / source`, mở đầu bằng callout `> [!summary] TL;DR`.
- 🔜 = khung (chưa viết nội dung) · ✅ = đã có nội dung.

> **Tip:** Dùng Obsidian Graph View để thấy toàn bộ liên kết giữa các note.
